site_name: ETC Measurement Starter

The ETC measurement starter was designed to start conductivity measurements inside the Thermal Constants Analyzer (Automation Master in Hot Disk software must be active)
![measurement_starter.PNG](../../assets/pics/measurement_starter.PNG)

- Select a folder where the measurement results should be stored. 
- To start a measurement click *Start Measurement* 
  - This starts the currently loaded schedule in the *Thermal Constants Analyzer* software
  - When the measurement finished click *Export Results*
    - When you have activated the *Start Log File Tracker* button in the main window measurement results will be automatically imported into your database and included in the live plot.
